Introduction

C++ is a general-purpose programming language that allows for efficient low-level memory manipulation, object-oriented design, and more. In this platform, we will cover key C++ concepts with examples.

Syntax

The syntax of a C++ program typically includes a `main` function, header files, and statements written with semicolons.

#include 
using namespace std;

int main() {
    cout << "Hello, C++!" << endl;
    return 0;
}

Input and Output

In C++, we can take input using `cin` and print output using `cout`.

#include 
using namespace std;

int main() {
    int num;
    cout << "Enter a number: ";
    cin >> num;
    cout << "You entered: " << num << endl;
    return 0;
}

Variables and Constants

Variables store data and constants are values that cannot be changed after initialization.

#include 
using namespace std;

int main() {
    const int MAX = 100;
    int var = 10;
    cout << "Variable: " << var << ", Constant: " << MAX << endl;
    return 0;
}

Data Types and Literals

C++ supports various data types like `int`, `float`, `char`, etc. Literals represent fixed values like `5`, `3.14`, etc.

#include 
using namespace std;

int main() {
    int a = 5;
    float b = 3.14;
    char c = 'A';
    cout << "Integer: " << a << ", Float: " << b << ", Char: " << c << endl;
    return 0;
}

Operators

C++ includes arithmetic operators, comparison operators, logical operators, etc.

#include 
using namespace std;

int main() {
    int a = 5, b = 3;
    cout << "Sum: " << a + b << endl;
    cout << "Product: " << a * b << endl;
    return 0;
}

Strings

Strings are sequences of characters in C++. They can be manipulated using various functions.

#include 
#include 
using namespace std;

int main() {
    string str = "Hello, C++!";
    cout << "String: " << str << endl;
    return 0;
}

Functions

Functions in C++ are blocks of code designed to perform specific tasks.

#include 
using namespace std;

void greet() {
    cout << "Hello from the function!" << endl;
}

int main() {
    greet();
    return 0;
}
