Introduction to C

C is a versatile programming language commonly used in operating systems, embedded systems, and other performance-critical applications.

Basic Syntax

A C program begins with a main function. Example:

#include 

int main() {
    printf("Welcome to C Programming!");
    return 0;
}

output : Welcome to C Programming!

Control Structures

Control structures allow decision-making and repetition:

Example: Check if a number is positive or negative

#include 

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);

    if (num > 0) {
        printf("Positive number.");
    } else {
        printf("Negative number.");
    }
    return 0;
}
input1   : 20
output1  : Positive number
input2   : -75
output2  : Negative number

Arrays

Arrays store multiple values of the same data type.

Example: Calculate the average of numbers

#include 

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        sum += numbers[i];
    }

    printf("Average: %d", sum / 5);
    return 0;
}

output : 30

Pointers

Pointers store the memory address of another variable.

Example: Swap two numbers using pointers

#include 

void swap(int *x, int *y) {
    int temp = *x;
    *x = *y;
    *y = temp;
}

int main() {
    int a = 5, b = 10;

    printf("Before swapping: a = %d, b = %d\n", a, b);
    swap(&a, &b);
    printf("After swapping: a = %d, b = %d", a, b);
    return 0;
}
